Features

Read all about the features of the calculator and how to use them properly

  • Player selection: Here you can select the number of players for the current game. It will get locked automatically after the "Add Round" button is pressed to avoid accidentally changing it.
  • The input table: Here you will input all the scores for the current round.
    • The setup row: The setup row contains an input field to set the number of points each player must earn to win the game. It will get locked automatically after the "Add Round" button is pressed.
    • Score remaining: This row shows the amount of points each player has remaining that they have to reach in order to win the game. It will updated every time the "Add Round" button is pressed.
    • Round X: This row shows the number of the round in the first column. After that, there are 3 input fields for each player to put the data of their throws in. Note, that it is not necessary to input a 0, but you can rather just leave the fields empty. The data is not case-sensitive. The format is as follows:
      • Bullseye Inner: BI
      • Bullseye Outer: BO
      • Single: 1 to 20
      • Double: D1 to D20
      • Triple: T1 to T20
    • Last Round: Shows the points earned in the last round. Will also show the points, if the player overshoots the target score.
  • The "Add Round" button: This button is locked by default, until a number of players is selected. Every time it is pressed, a round will get registered. This means, that the current round will get added to the history and the input fields will get cleared again.
  • The history: The history is the box at the bottom. It will automatically have a new row added for every round registered. The format looks as follows:
    • Round X: Shows which round the entry is from.
    • First row of the player data: The first row of the playerdata shows a summary of the calculated score based in the throw inputs. The format is as follows: score1 + score2 + score3 = total.
    • Second row of the player data: The second row of the playerdata shows the starting amount of points the player had to reach and the amount of points the player has to reach after this round has passed. The format is as follows: startScore ====> scoreAfterThisRound.
    • Average: The average shows the average scores per round. The format is as follows: averagePointsThisRounds (averagePointsPerThrow).
